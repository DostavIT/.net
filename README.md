# .net клиент для сервиса [DostavIT](http://dostav.it/)
Сервис DostavIT предоставляет информацию о крупнейших компаниях-грузоперевозчиках по России и СНГ, а так же позволяет узнавать актуальную информацию о ценах/сроках доставки Вашего груза.

.net клиент для сервиса грузоперевозок [DostavIT](http://dostav.it/) - это portable библиотека, которую можно подключить к .net проекту, чтобы получить доступ ко всем функциям сервиса. Библиотека может использоваться с .net framework версии 4.5 или выше. Поддерживается работа на Windows 8 или выше, а так же на мобильных телефонах с Windows Phone 8.1

### Использование библиотеки
Для использования клиента необходимо собрать библиотеку и подключить её к своему проекту. После этого станет доступным статичный класс `DostavItService`. Он позволяет:
 * `GetCarriersAsync()` - получить информацию о поддерживаемых компаниях-грузоперевозчиках
 * `GetCarrierAsync(carrier)` - получить информацию о конкретной компании-грузоперевозчике, передав её идентификатор в качестве параметра `carrier`
 * `GetRatesAsync(fromCity, toCity, lengthCm, widthCm, heightCm, weightKg, costRub, shipDate)` - получить информацию о стоимости и сроках доставки груза из одного города в другой. Названия городов указываются на русском языке в соответствии с их официальным наименованием и передаются в качестве параметров `fromCity` и `toCity` - города отправки и доставки соответственно. `lengthCm`, `widthCm`, `heghtCm` - длина, ширина и высота груза в сантиметрах. `weightKg` - вес груза в килограммах. `costRub` - декларируемая стоимость содержимого в рублях. `shipDate` - необязательный параметр, дата планируемой отправки груза, исходя из которой будет рассчитываться ориентировочная дата доставки. Если этот параметр не указан, будет использоваться сегодняшняя дата.
 * `GetRatesAsync(carrier, fromCity, toCity, lengthCm, widthCm, heightCm, weightKg, costRub, shipDate)` - работает аналогично методу, описанному выше, но возвращает результаты только для одной компании-грузоперевозчика с идентификатором `carrier`

**Обратите внимание!** Параметр `carrier` - это строковый ключ, используемый в рамках сервиса [DostavIT](http://dostav.it/) для идентификации компаний-перевозчиков. Он не совпадает с названием компании. Для получения полного списка перевозчиков с их идентификаторами рекомендуется использовать метод `GetCarriersAsync()`

### Структура данных
`CarrierInfo` содержит информацию о компании-перевозчике
 * `Name` - идентификатор компании в сервисе [DostavIT](http://dostav.it/)
 * `FullName` - официальное название компании
 * `Url` - адрес сайта компании
 * `LargeLogoUrl` - ссылка на логотип компании
 * `Services` - список услуг по грузоперевозке, предоставляемых данной компанией.

`DeliveryService` - содержит информацию об услуге по грузоперевозке, предоставляемой компанией
 * `Name` - идентификатор услуги
 * `FullName` - полное название услуги

`Rate` - содержит информацию о стоимости и сроках доставки груза
 * `Carrier` - идентификатор компании-грузоперевозчика
 * `Service` - идентификатор услуги по доставке груза
 * `ShipDate` - планируемая дата отправки груза
 * `DeliveryDate` - дата, когда груз будет доставлен. Данное поле может быть пустым, если дату доставки не удаётся рассчитать
 * `Cost` - стоимость доставки с указанием валюты
 * `Insurance` - стоимость страхования груза с указанием валюты
